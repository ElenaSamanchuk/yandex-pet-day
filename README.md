## Что внутри

| Слой | Содержание |
|------|------------|
| **Figma** | 1440 + 360, UX/UI Kit, Components, Play ▶, Dev · States |
| **Код** | Vite 6, Tailwind v4, GSAP — интерактивный HTML-прототип |
| **Docs** | `docs/hod-mysli.md`, `docs/ux-ui-kit.md`, `docs/figma-handoff.md` |

## Интерактив

- FAQ accordion (+/−)
- Форма: офлайн / онлайн → смена hint, скрытие «Приду с питомцем»
- Modal «Остались вопросы»
- Toast после регистрации
- Burger-menu на 360 px

## Сетка

- Viewport desktop **1440** → контент **1200** (gutter 120 px)
- Mobile **360** → контент **320** (gutter 20 px)

## Запуск локально

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```
